## Get Started

### Prerequisites
- Java 17+
- Maven

### Running the project

1. Clone the repository

```bash
git clone https://github.com/wesleybertipaglia/sudoku.git
```

2. Navigate to the project directory

```bash
cd sudoku
```

3. Build the project

```bash
mvn clean install
```

4. Run the app

```bash
mvn exec:java -Dexec.mainClass="com.wesleybertipaglia.Main"
```
